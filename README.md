# Superstore Analysis (Power BI)

## 📊 Superstore Sales Analysis (Power BI)

This project analyzes retail sales data using Power BI to uncover business insights.

### 🔧 Tools Used
- Power BI
- DAX
- Data Cleaning

### 📌 Key Insights
- Identified top-performing regions and categories
- Analyzed profit trends and loss-making segments
- Created interactive dashboards with KPIs

### 📷 Dashboard Features
- Sales Overview
- Profit Analysis
- Regional Performance
